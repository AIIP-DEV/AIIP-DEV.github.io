사용자 현황

전체 사용자에 대한 정보와 현황을 조회하고, 사용자 추가/삭제/잠금/비밀번호 초기화 기능을 포함합니다.

사용자 상태​

  • Active (정상): 계정이 활성화 상태로 정상 사용중인 사용자 상태입니다.
  • Locked (잠김): 사용자 로그인에서 패스워드를 5회이상 입력 오류로 인해 잠금 상태인 계정입니다.
  • Blocked (차단): 서비스 부정사용 등의 이유로 관리자 권한을 가진 사용자를 통해 계정이 서비스를 사용할 수 없는 상태인 사용자입니다.
  • Deleted (삭제): 관리자 권한 사용자가 더이상 서비스를 사용하지 않는 사용자 계정을 삭제한 상태입니다.
정보
  • Locked 잠금 상태 계정의 경우 관리자 권한의 사용자가 활성화 할 수 있으며, 활성화 시 초기 패스워드 값으로 설정됩니다. 초기 패스워드의 경우 환경에 따라 지정된 패스워드가 있습니다.
  • Blocked 차단된 계정의 경우 관리자 권한의 사용자를 통해 Active 상태로 변경 할 수 있습니다.

사용자 관리​

사용자 목록 [Action] 컬럼의 버튼을 통해 개별 사용자에 대한 관리 기능을 제공합니다.

  • 정상 사용자의 경우 Block 상태로 변경
  • 차단된 사용자를 Active 상태로 변경
  • 사용자 정보 수정
  • 사용자 삭제
  • 잠김 상태인 사용자의 비밀번호 초기화
