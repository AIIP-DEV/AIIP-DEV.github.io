사용자 등록

현재 버전의 SSO 서비스에서 사용자 추가는 관리자 권한을 가진 사용자를 통해서만 신규로 등록이 가능합니다.

계정등록​

단일 사용자 등록의 경우 사용자 목록 상단 [계정등록] 버튼을 통해 추가됩니다. ID/Email 항목은 기존 등록된 사용자와 중복 등록될 수 없으며, 적절한 권한을 부여하고 해당하는 사용자 그룹을 할당하여 등록합니다.

정보

신규 계정의 경우 패스워드는 초기값으로 설정됩니다. 초기 패스워드의 경우 환경에 따라 지정된 패스워드가 있습니다.

템플릿을 이용한 다중 사용자 등록​

다중 사용자 등록은 목록 상단 [계정 일괄 등록 (CSV)] 버튼을 통해 등록 할 수 있습니다.

  1. 일괄 등록의 경우 SSO 서비스에서 제공하는 템플릿에 따라 항목을 편집하고 편집된 파일을 업로드합니다.
    • 사용자 권한 : 0=System Admin, 1=Service Admin, 2=User
  2. 서비스에서는 업로드된 파일의 유효성 검사를 진행하고 잘못된 항목이 포함된 경우 각 항목별로 메시지를 노출합니다.

전체 항목에 대해서 유효한 경우 정상적으로 [저장] 버튼이 활성화 되며 등록이 가능합니다.

정보

다중 계정 등록시에도 모든 계정에 대해서 패스워드는 동일하게 초기값으로 설정됩니다.
초기 패스워드의 경우 환경에 따라 지정된 패스워드가 있습니다.
