모델 레지스트리 목록

  • 모델 레지스트리에 속해 있는 모델의 상태를 보여주는 기능을 제공합니다.
  • 모델 자원배포 생성 및 수정, 모델 등록을 수행합니다.
​

모델 레지스트리 화면​

모델 레지스트리 목록 화면입니다.

​
​

1. 공통 GNB​

  • 공통 GNB 부분입니다.
    • 대시보드 - 공통 GNB 화면으로 이동합니다.
​

2. 모델 레지스트리 UI 선택​

  • 카드/테이블 형태의 목록 중에서 선택할 수 있는 기능입니다.
​

3. 검색​

  • 문자열로 모델명을 검색하는 기능입니다.
  • 생성자, 서빙중인 모델을 기준으로 필터링 할 수 있습니다.
​

4. 모델 리스트 정렬​

  • 최신 업데이트 순 / 제목 오름차순 / 제목 내림차순 으로 모델 리스트를 정렬하는 기능입니다.
​

5. 모델 등록​

  • 새로운 모델의 버전 1 을 등록하는 기능(모델 파일 업로드/모델 패키지 업로드)입니다.
  • 모델 등록 기능은 다음을 참조하시면 됩니다.
    • 모델 등록 화면으로 이동합니다.
​

6. 모델 배포자원 정보​

모델 배포 시 사용할 자원을 설정하는 화면입니다.

  • 생성된 배포자원이 없으면 배포자원을 생성할 수 있습니다.
  • 배포자원이 생성되어 있으면 배포자원 정보가 표시됩니다.
  • Small, Medium, Large, 직접 설정으로 배포자원을 설정할 수 있습니다.
  • 생성된 배포자원은 수정 기능을 이용하여 수정할 수 있습니다.
​

7. 배포 이력​

프로젝트에서 발생한 배포 이력을 표시합니다.

  • 항목은 날짜 별로 구분되어 표시됩니다.
  • 표시되는 이력 항목
    • 모델 등록/삭제
    • 모델 버전 등록/삭제
    • Canary 모델 배포/중지
    • 모델 서빙 설정 변경
    • 모델 프로모션
    • 모델 서빙 중지/일시 중지/재시작
​

8. 모델 정보​

  • 모델 레지스트리명, 모델명, 모델상태를 보여줍니다.
​

9. 모델 버전 추가 및 삭제​

  • 모델 버전 추가 또는 삭제를 실행합니다. 단, 서빙중인 모델은 삭제할 수 없습니다.
  • 모델 버전 추가는 다음을 참조하시면 됩니다.
    • 모델 상세 - 버전 현황 - 2. 모델 버전 추가 화면으로 이동합니다.
​

10. AI Runtime 단독모드일 때 모델 레지스트리 정보/삭제​

AI Runtime 단독모드일 때만 적용되는 기능입니다.

  • 모델 레지스트리 정보를 수정하거나 삭제할 수 있습니다.
