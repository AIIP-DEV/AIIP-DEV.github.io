버전 현황

  • 모델의 버전을 추가하고 서빙 사작 및 모델 버전 별 상태를 확인할 수 있는 화면입니다.
​

버전 현황 화면​

​

1. 모델 상세 공통​

  • 모델 상세 공통 부분입니다.
    • 모델 상세 화면으로 이동합니다.
​

2. 모델 버전 추가​

모델의 버전들을 표시하고 모델 버전 등록으로 버전을 추가할 수 있습니다.

​

모델 버전 추가 버튼을 클릭하면 나타나는 모델 버전 등록 화면입니다.

  • 필수 항목들을 설정하고 등록하기 버튼을 클릭하면 버전이 추가 됩니다.
  • 설정 방법은 모델 등록 화면과 유사합니다.
    • 모델 등록 화면으로 이동합니다.
​

모델 버전이 추가되면 모델 빌드가 시작 됩니다.

​

3. 모델 상태​

모델 등록 후 모델 상태는 빌드중 / 배포가능 / 빌드실패 로 표시 됩니다. 모델 상태가 배포가능이면 서빙 시작 버튼이 표시 됩니다.

4. 서빙 / 서빙 상태​

서빙 동작을 실행할 수 있고 서빙 상태를 표시합니다.

​

4-1. 서빙 시작​

서빙 중인 모델이 없는 경우에는 서빙 시작 버튼을 클릭하여 서빙을 시작할 수 있습니다.

  • (1) 서빙 버전
    • 서빙할 모델의 버전입니다.
  • (2) Concurrency
    • 서빙할 모델의 Concurrency 값 입니다.
  • (3) 자원설정 - 배포 자원
    • 요청한 자원 크기 / 배포 자원에서 사용 가능한 크기 를 각 자원 별로 나타냅니다.
  • (4) 자원설정 - 자원 타입
    • Small, Medium, Large 또는 직접 설정을 선택하여 자원을 요청할 수 있습니다.
    • 자원의 최대 설정값에 따라 요청한 자원 크기가 달라집니다.
  • (5) Replica
    • Replica 값의 최소 ~ 최대를 설정합니다. Replica 최대 값에 따라 요청한 자원 크기가 달라집니다.
    • Sidecar - Replica 최대 값 만큼 CPU, Memory 자원이 추가로 할당되어 '자원설정 - 배포 자원'에 표시 됩니다. (안내 문구 참조)
    • Zero scale - Replica 최소 값을 0으로 설정하면 서비스를 사용하지 않을 때는 Pod 가 떠있지 않습니다. (안내 문구 참조)
  • (6) GPU
    • GPU 값을 설정합니다.
​

4-2. 서빙 상태​

서빙 시작 후 서빙 상태를 WAITING 으로 표시합니다.

​

서빙 상태가 RUNNING 으로 표시되면 예측 요청 및 Canary 배포가 가능합니다.

​

4-3. 에러 정보 팝업​

서빙 상태가 ERROR로 표시될 때 ERROR를 선택하면 에러정보 팝업이 띄워서 에러 내용을 보여줍니다.

5. Canary 배포 / Canary 배포 상태​

  • 서빙중인 모델이 존재하는 상태에서 새 버전의 모델을 배포하는 경우에 Canary 배포 기능을 사용할 수 있습니다. Canary 배포를 사용하면 현재 서빙중인 모델과 새 버전의 모델을 동시에 서빙하여 테스트하고 새 버전의 모델을 프로모션 할 수 있습니다.
  • 예) 예측 요청 비율을 Default 모델 90% + 새 버전의 모델 10% 로 지정하면 사용자의 요청 중 90%는 Default 모델, 나머지 10%는 새 버전의 모델로 보내져 처리되고, 새 버전의 모델에서 이상 징후가 발견되지 않으면 비율을 점차 늘려서 최종적으로 새 버전의 모델로 완전히 교체 가능합니다.
​

Canary 배포 추가 버튼을 클릭합니다.

  • (1) Canary Traffic Percent
    • Default 버전과 Canary 버전의 서비스 간 Canary Traffic Percent 를 설정합니다.
  • (2) Default 서빙 자원
    • Default 버전 서비스의 서빙 설정으로 수정이 불가능 합니다.
  • (3) Canary 서빙 자원 설정
    • Canary 버전 서비스의 서빙 자원을 설정합니다. (4-1. 서빙 시작 참조)
​

6. 변경 일시​

  • 해당 버전의 상태가 변경된 일시 입니다.
​

7. 모델 버전 상세 정보​

버전을 선택하면 해당 버전의 모델의 상세 정보를 표시합니다.

​

8. 컨텍스트 메뉴​

모델 등록 정보 화면입니다.

  • 모델 등록 정보
    • 모델의 등록 정보를 표시합니다.
  • 워크플로우 조회
    • 모델의 워크플로우로 이동합니다.
  • 모델 버전 삭제
    • 모델의 버전을 삭제합니다.
