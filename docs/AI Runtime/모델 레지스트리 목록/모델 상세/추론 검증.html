추론 검증

  • 서빙 중 모델의 성능을 계산 및 후보 모델과의 성능을 비교 검증할 수 있는 화면입니다.
​

추론 검증 화면​

모델 상세 화면에서 추론 검증 탭을 선택했을 때 출력되는 화면입니다.

​

1. 모델 상세 공통​

  • 모델 상세 공통 부분입니다.
    • 모델 상세 화면으로 이동합니다.
​

2. 시간 설정으로 추론 검증 이력 검색​

  • 최근 3시간 / 최근 2시간 / 최근 1시간 / 직접 설정 등의 시간 설정으로 추론 검증 이력을 검색할 수 있습니다.
​

3. 이력 번호로 추론 검증 이력 검색​

  • 검색창에서 이력 번호 또는 테스트 ID로 추론 검증 이력을 검색할 수 있습니다.
​

4. 새로 고침​

  • 선택된 검색 조건에 맞추어 새로 고침을 수행합니다.
​

5. 검증 등록​

추론 검증 등록을 위한 팝업입니다.

  • 모델이 서빙중일 때 검증등록 버튼이 활성화 됩니다.
  • 파일을 업로드하는 위치이며, 파일 업로드 시 파일 명을 노출하고 추론 검증 조건에 위배되는 파일일 때 에러 메세지를 노출합니다.
  • 업로드한 csv 파일을 보여줍니다.
  • 파일 업로드 시 해당 모델의 검증을 위한 지표 목록을 표시하며 이중 하나를 대표 지표로 선택할 수 있습니다.
  • Canary 버전이 서빙중일 때와 그렇지 않을 때의 검증 대상 버전 UI가 다르게 표시됩니다.
  • 후보 버전을 선택했을 때 후보 버전 서빙 자원을 설정할 수 있는 UI가 노출됩니다.
​

6. 추론 검증 이력 항목​

  • 이력 번호 / 요청 시간 / 대표 지표 / 기본 버전 점수 / 후보 버전 점수 / 소요 시간 / 테스트 ID / 결과 를 알려 줍니다.
​

7. 추론 검증 결과​

추론 검증 결과를 조회할 수 있는 팝업입니다.

  • 이력 정보, 추론 결과를 보여주고 버전 별 지표 점수를 보여줍니다.
