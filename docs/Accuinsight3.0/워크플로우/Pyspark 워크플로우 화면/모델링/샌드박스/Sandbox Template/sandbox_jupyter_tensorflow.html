Tensorflow Template

Sandbox Template의 손쉬운 사용을 위해 주피터 노트북 예제파일이 준비되어 있습니다.

주피터 노트북 예제 파일을 생성하여 Sandbox Template 사용법을 확인해보겠습니다.

해당 Tensorflow 주피터 노트북 예제파일은

iris tabular 데이터를 이용하여 Tensorflow 모델을 훈련하고

Accuinsight+ 3.0에 배포하는 것을 목표로 합니다.

예제 주피터 노트북 파일 생성 방법​

  1. Jupyter lab 에 접속
  2. 메뉴바 > File > Template
    • accuinsignt+ 아래에 accuinsight+/airuntime_tensorflow.ipynb 를 선택한 후, Go 를 선택합니다.
    • 아래와 같이 Untitled.ipynb 파일이 생성된 것을 볼 수 있습니다.

​

데이터셋 노드 연결하기​

아래와 같이 데이터셋 노드를 샌드박스 노드에 연결하면

Jupyter Lab에서 해당 데이터셋을 사용할 수 있습니다.

​

예제의 실행을 위해 잠시 샌드박스 화면에서 나가 데이터셋 노드를 연결해보겠습니다.

워크플로우 화면 우측에서 데이터 탭을 클릭한 뒤 데이터 가져오기를 클릭합니다.

​

상단에서 Asset 탭을 클릭하면 미리 준비된 iris tabular 데이터셋을 볼 수 있습니다.

iris 데이터셋의 체크박스를 클릭한 뒤, 등록 버튼을 누릅니다.

​

우측 상단의 +추가 버튼을 클릭하면 데이터셋 노드가 생성됩니다.

​

생성된 데이터셋 노드를 드래그하여 Jupyter Lab 노드 근처로 끌어당기면

화살표가 생성되어 데이터셋노드와 Jupyter Lab 노드가 연결됩니다.

그러면 Jupyter Lab 노드를 우측 하단의 Running에서 Stop으로 바꾼 뒤, 종료되면 다시

Running을 클릭하여 노드를 재시작해줍니다.

​

이제 그리고 다시 Jupyter Lab으로 접속합니다.

​

데이터 가져오기​

데이터 가져오기 코드를 실행하면 연결한 데이터셋 노드의 데이터를 사용 준비합니다.

만약 여러개의 데이터셋 노드를 샌드박스에 직접적으로 연결했다면 첫 번째 데이터셋 노드만을 사용합니다.

아래 코드를 실행하면 데이터 가져오기를 수행하고, 가져온 데이터의 정보를 확인할 수 있습니다.

데이터는 Pandas DataFrame의 형태로 가져옵니다.

만약 iris가 아닌 다른 데이터셋을 사용하고 있다면 현재 모델이 tabular 데이터를 사용하고 있다는 점을 감안하여 코드를 수정하면 됩니다.
