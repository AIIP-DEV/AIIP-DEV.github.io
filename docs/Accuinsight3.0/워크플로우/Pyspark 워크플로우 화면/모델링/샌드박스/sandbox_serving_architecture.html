Sandbox Serving Architecture

샌드박스에서 사용자가 저장한 모델은 Docker Image로 빌드되어

Accuinsight+ 3.0 환경에 모델 예측이 가능한 형태로 배포됩니다.

샌드박스에서 작업한 모델이 배포되는 구조는 아래 도식과 같습니다.

사용자가 샌드박스에서 모델을 추출하고 레지스트리에 등록하는 작업을 수행하면,

이 샌드박스는 Docker image로 빌드되어 레지스트리에 업로드 됩니다.

이후 배포하게되면 Accuinsight+ 3.0 클러스터에 배포되어 모델 예측을 REST API 형태로 받을 수 있는 웹 서버로서 기능합니다.

이후 예측 요청 API를 사용하고 이 요청이 배포된 모델에 도달한 후 모델 예측 결과값을 받는 구조는 아래와 같습니다.

웹 서버가 생성되며 부여받은 주소에 REST API 요청이 Request Body의 형태로 전달되고

전처리를 거쳐 모델의 예측결과가 생성된 후, 후처리를 거쳐 아래 Response Body와 같은 형태로 돌아오게 됩니다.

우측의 예시코드는 샌드박스 내에서 확인할 수 있습니다.
