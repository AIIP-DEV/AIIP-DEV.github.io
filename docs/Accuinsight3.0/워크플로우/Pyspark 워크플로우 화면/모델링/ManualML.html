ManualML

  • 데이터를 등록하고 Estimator와 Hyper Parameter을 직접 설정하여 모델링을 실행할 수 있습니다.

  • 지도학습 방법 - 분류, 예측

    • 분류 : 원하는 대상을 몇가지 값으로 분류할 수 있습니다.(ex)이탈고객분류, 타이타닉 생존자 분류
    • 예측 : 임의의 대상을 특정범위로 예측할 수 있습니다.(ex)뉴욕시 집값 예측, 비가 올 확률 예측

ManualML생성​

빈 Workflow에서 ManualML 생성​

  1. 워크플로우 작업 선택에서 빈 워크플로우를 선택한 후 '다음' 버튼을 클릭하여 워크플로우를 생성합니다.

  2. 워크플로우 내에서 Data를 가져온 뒤, Data노드를 클릭하고 노드메뉴에서 모델링 > ManualML 노드를 선택합니다.

  3. 추가 버튼을 눌러 ManualML을 생성하면 ManualML노드가 준비됩니다.

ManualML 준비​

  • ManualML 노드가 workflow에 추가되면 데이터를 읽어서 ManualML을 하기 위한 준비를 합니다.

    1. 준비가 완료되면 실험을 추가할 수 있는 실험 리스트로 이동합니다.

    2. 실험 추가 버튼을 누르면 실험이 추가됩니다.

ManualML 실행​

  • 추가된 실험의 우측 실행 버튼을 눌러 실험 이름, 분석 기법, 타깃, 지표를 설정하여 실행할 수 있습니다.

  • ManualML 설정 우측 상단 실행 버튼으로 실행할 수 있습니다.

    해당 팝업에 나타는 값에 대한 설명은 아래와 같습니다.

    • 분석 기법
      • 분류 : 분류에 대한 머신러닝을 적용하실 수 있습니다.
      • 회귀 : 회귀(예측)에 대한 머신러닝을 적용하실 수 있습니다.
    • 타깃 : 데이터 중 분류/회귀(예측)하고자 하는 필드를 선택하실 수 있습니다.
    • 지표 : 평가지표를 선택하실 수 있습니다.(ex)Accuracy, F1 score, Recall, Precision 등

ManualML 설정​

  • 실험의 상세정보 페이지나 해당 실험의 컨텍스트 메뉴의 실험 수정을 클릭합니다.
  • 아래와 같은 내용중에 ManualML 설정을 고르실 수 있습니다.
    • 기본 설정
    • 학습옵션 선택
    • 분할 (Optional)
    • 샘플링 (Optional)
    • 고급 설정 (Optional)

기본 설정​

우측 패널에 나타는 값에 대한 설명은 아래와 같습니다.

  • 분석 기법
    • 분류 : 분류에 대한 머신러닝을 적용하실 수 있습니다.
    • 회귀 : 회귀(예측)에 대한 머신러닝을 적용하실 수 있습니다.
  • 타깃 : 데이터 중 분류/회귀(예측)하고자 하는 필드를 선택하실 수 있습니다.
  • 지표 : 평가지표를 선택하실 수 있습니다.(ex)Accuracy, F1 score, Recall, Precision 등

학습옵션 선택​

  • Data Columns 탭을 선택한 후 사용할 컬럼을 체크할 수 있습니다.
    • 데이터 타입은 아래와 같은 내용을 선택하실 수 있습니다.
      • OBJECT
      • INT64
      • FLOAT64
      • DATETIME64
      • TEXT
    • 결측치 대체 방법은 아래와 같은 방법을 제공합니다.
      • NONE
      • DROP
      • MOST_FREQUENT
      • UNKNOWN
      • MISSFOREST
      • KNN
      • XGB
      • LGBM
    • 데이터에 대한 변형은 아래와 같은 방안을 제공합니다.
      • NONE
      • LOG
      • SQUARED_ROOT
      • SQUARE
      • BOX_COX_TRANSFORMATION
      • YEO_JOHNSON_TRANSFORMATION
      • ADD_LAGS
    • 이상치 제거 : 이상치가 있을 경우 제거 유무를 선택할 수 있습니다.

분할 (Optional)​

  • 데이터가 부족하거나 많을 때는 분할을 이용하여 샘플링을 사용하실 수 있습니다.
    • Resampling Strategy : 데이터가 부족하거나 많을 때는 아래와 같은 방식으로 샘플링을 사용하실 수 있습니다.
      • Holdout : Train(훈련용)/Validation(점검용)/Test(테스트용)로 나누어서 테스트하는 방식입니다.
      • CV : Cross Validation(교차 검증)으로 데이터셋을 여러 등분으로 나누어 Train(훈련용)/Validation(점검용) 데어터셋을 교대로 바꿔가면서 테스트하는 방식입니다.
    • Split Ratio : 데이터셋을 분리할 비율을 나눌 수 있습니다.
    • Test 사용 : Test데이터셋의 사용 유무를 선택할 수 있습니다

샘플링 (Optional)​

  • 데이터셋의 데이터가 백만건 이상일 경우 강제로 샘플링을 사용하여 모델을 훈련하게 됩니다.
    그 외의 경우 샘플링 유무를 선택하여 진행하실 수 있습니다.
    • Sampling Target : 샘플링 사용 유무 혹은 종류를 선택합니다.
      • 없음
      • ALL
      • TRAIN
    • Sampling Type
      • Size
      • Ratio
      • Augmentation
    • Sampling Size: 데이터셋의 대상 데이터 갯수를 100개에서 150개까지로 제한합니다.
    • Class Balancer : 데이터셋이 데이터가 편중되어서 한쪽이 많거나 적을 때 데이터셋의 분포가 적당하도록 Over-Sampling(증가)하거나 Under-Sampling(감소) 기능을 제공합니다.

ManualML 진행​

  • ManualML 노드를 더블 클릭하여 실험 리스트를 열면 진행상태(진행중, 완료)와 Valid를 확인할 수 있습니다.

  • 완료된 실험의 모델 생성 버튼을 클릭하여 배포를 진행합니다.

ManualML 완료​

  • 배포가 완료되면 실험 리스트에 배포 완료가 표시됩니다.

  • 배포가 완료되면 워크플로우에 모델 노드가 생성됩니다.

ManualML 노드 옵션​

  • ManualML 노드를 마우스 우클릭하면 manualml 노드 옵션이 나타납니다.
  1. ManualML 설정하기
    • ManualML 각 실험과 결과를 조회하는 화면으로 이동합니다.
  2. 이름 바꾸기
    • 노드 이름을 변경할 수 있습니다.
  3. 즐겨찾기에 추가
    • 우측 패널의 즐겨찾기에 ManualML 노드를 추가합니다.
  4. 삭제
    • ManualML 노드를 삭제합니다.

ManualML 실험 리스트​

  • ManualML 노드를 마우스 더블 클릭 또는 우클릭 하여 manual 노드 옵션에서 ManualML 설정하기를 선택합니다.

실험 설정 값​

  • 해당 실험의 기본 정보를 조회할 수 있으며 실행하지않은 실험은 실험을 수정할 수 있습니다.

모델 노드​

  • 실험 리스트에서 모델 노드를 배포할 수 있습니다.
  1. 모델 노드
    • ManualML이 완료된 후 생성된 모델 노드입니다.
  2. 모델 노드 옵션
    • 모델 레지스트리 등록
      • 새 모델 등록
        • AI Runtime에 새 모델을 등록합니다.
        • 등록할 모델이름 입력하고 중복확인 후 확인을 클릭하여 새 모델을 등록합니다.
      • 새 버전으로 등록
        • AI Runtime에 배포된 모델이 있을 경우 해당 모델의 새 버전으로 모델을 등록할 수 있습니다.
        • 배포된 모델 중 하나를 선택한 후 확인을 클릭하여 새 버전으로 모델을 등록합니다.
    • 이름 바꾸기
      • 모델 노드의 이름을 변경합니다.
    • 삭제
      • 모델 노드를 삭제합니다.
  3. 예측용 Workflow 생성
    • 현재 워크플로우 이름에 Predict prefix를 붙여서 예측용 워크플로우를 생성합니다.

예측용 Workflow​

  • 예측용 워크플로우는 이미 훈련을 마친 모델을 사용하여 실제 데이터를 넣어서 분류/예측을 실행시키고 싶을 때 사용하는 워크플로우입니다.

  1. 워크플로우 목록에서 예측용 워크플로우 선택하여 불러옵니다.

  2. 데이터 가져오기를 클릭하여 가져올 데이터를 선택합니다.

  3. 데이터 가져오기가 완료된 후 모델 노드를 선택하고 우측 패널에서 예측 버튼을 클릭합니다.

  4. 예측 완료 후 예측 결과 데이터 노드가 생성됩니다.

  5. 예측 결과 데이터 노드를 선택하고 우측 패널에서 정보를 확인합니다.
