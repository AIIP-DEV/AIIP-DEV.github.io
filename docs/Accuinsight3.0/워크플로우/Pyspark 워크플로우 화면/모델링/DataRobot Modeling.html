DataRobot Modeling

DataRobot Modeling​

DataRobot에서 직접 모델링을 수행하기 위해 학습용 데이터를 DataRobot 사이트에서 직접 등록하여 나온 모델링 결과를 AccuInsight+ 3.0에 배포할 수 있습니다.

DataRobot에 데이터 내보내기​

DataRobot Modeling를 위해 AccuInsight+ 3.0의 Workflow에서 데이터 노드를 선택한 후 마우스를 우클릭하여 DataRobot에 내보내기를 진행합니다.

주의

DataRobot에 내보내기를 진행하지 않은 데이터 노드에서는 DataRobot Modeling 노드가 비활성화 됩니다.

정보

DataRobot에 내보내기를 위한 사용 설정은 연동 관리를 참고해 주시기 바랍니다.

​

DataRobot AI Catalog 이동 방법​

DataRobot에 내보내기 완료 후 나오는 팝업에서 확인을 눌러 이동하시거나



Datarobot Modeling 노드를 생성하시고



Datarobot Modeling 우측패널에 있는 파란색 글씨로 되어 있는 DataRobot AI Catalog를 클릭하시면 됩니다.



​

DataRobot AI Catalog에서 프로젝트 생성​

DataRobot AI Catalog을 통한 모델링 수행과정을 진행하기 위해 먼저 DataRobot AI Catalog에서 프로젝트를 생성해야 합니다. 자세한 내용은 DataRobot 가이드(https://datarobot.mv-awsstag.com/docs/)의 내용을 참고하시기 바랍니다.

프로젝트 생성

모델링 시작

​

DataRobot 프로젝트 선택​

DataRobot에서 프로젝트 생성 후 AccuInsight+ 3.0의 Workflow에 있는 Datarobot Modeling 노드 우측패널에 프로젝트 선택을 누르게되면



DataRobot AI Catalog에서 생성한 프로젝트 목록 중 하나를 선택하여



추가 버튼을 선택하면 선택한 DaraRobot 프로젝트의 모델링 작업이 완료되는 것을 기다립니다.

​

DataRobot 모델링 배포​

DataRobot에서 진행한 모델링을 리더보드로 확인하고, 해당 모델링을 AccuInsight+ 3.0의 Workflow에 모델 노드로 배포가 가능합니다.

리더보드 확인​

모델링이 완료 된 프로젝트를 선택한 Datarobot Modeling 노드를 더블클릭 혹은 우측 패널의 리더보드 버튼을 누르면 리더보드 화면으로 이동합니다.



리더보드 화면 구성​

1. 뒤로가기 버튼
2. 선택한 DataRobot 프로젝트를 새 창으로 열어 이동
3. 선택한 모델의 Datarobot 화면으로 새 창으로 열어 이동
4. AccuInsight+ 3.0의 Workflow에 모델 배포

모델 배포​

리더보드 화면에 배포 버튼을 누르면 AccuInsight+ 3.0으로 배포가 진행되며, 완료되면 workflow에 모델노드가 생성 됩니다.



정보

배포를 통해 생성된 모델 노드는 AutoML처럼 예측용 Workflow를 생성할 수 있습니다. 자세한 내용은 AutoML을 참고해 주시기 바랍니다.
