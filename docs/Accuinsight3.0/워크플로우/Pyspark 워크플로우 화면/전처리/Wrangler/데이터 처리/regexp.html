regexp

Definition​

선택한 대상 컬럼의 문자열에서 정규표현식에 기반하여 매칭되는 문자를 추출하여 새로운 컬럼으로 저장합니다.

정보

regexp는 신규 컬럼이 추가됩니다.

Property​

  1. newColumn : 추출한 문자가 포함될 새로운 컬럼명
  2. targetColumn : 대상 컬럼 선택
  3. regexp : 정규 표현식
  4. group_number : 사용할 표현식의 순서를 입력

Example​

[Gender] 컬럼에서 F, M 문자를 추출합니다.
