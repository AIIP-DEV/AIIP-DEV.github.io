기본 동작

Wrangler 노드는 워크플로우 [데이터 기본처리]에서 제공하는 다양한 노드를 실시간으로 확인하며 여러 노드를 하나의 노드로 구성합니다. 제공하는 데이터 처리는 항목 선택시 실시간으로 변경되는 항목과 사용자 입력으로 처리되는 항목으로 구분되며 컬럼 추가 또는 특정 항목에 대해서는 데이터 처리시 지정한 컬럼들의 데이터가 변경됩니다.

Wrangler 노드에서 제공하는 데이터 처리​

  1. 셀 단위 전처리

    • 공백제거
    • 찾아 바꾸기
    • 누락 채우기
    • 문자열 첨가

  2. 열 단위 전처리

    • 데이터 타입 변경하기
    • 소팅
    • 열 나누기
    • 문자열 추출하기 isin
    • 문자열 추출하기 substr
    • withColumn
    • Keep
    • 삭제

  3. 행 단위 전처리

    • 값에 따른 필터링
    • 상위 N% 필터링
    • Limit
    • 열 중복 제거
    • 전체 중복 제거
    • 열 결측치 제거
    • 전체 결측치 제거

  4. 더 많은 랭글러 기능

    • 타임스템프 추가
    • 날짜 포멧 변환
    • 정규식
    • 시간변경

기본 동작 방법​

  1. 데이터 요소에서 변환할 컬럼에서 컨텍스트 메뉴를 통해 원하는 데이터 처리를 확인합니다.
  2. 수행할 데이터 처리 항목을 선택하면 선택한 데이터 처리에 대해 미리 구성된 처리에 필요한 속성값을 지정합니다.
  3. 속성값을 변경하면 실시간으로 데이터셋 패널에서 변경된 값이 적용된 데이터를 확인할 수 있습니다.


다중 컬럼 선택​

Wrangler 노드는 단일 컬럼 데이터 처리 외에 다중 컬럼을 선택을 통한 데이터 처리를 제공합니다.

  • 첫번째 컬럼 선택 후 [shift] 키를 누른 상태에서 다른 컬럼 선택시 여러 컬럼을 동시에 선택할 수 있습니다.
  • 순차적이지 않은 여러 컬럼을 선택 할 경우 [ctrl] 키를 누른 상태에서 원하는 컬럼을 선택하여 동시에 여러 컬럼을 선택하고 컨텍스트 메뉴를 활성화 하여 원하는 데이터처리를 선택합니다.

적용된 항목 삭제​

Wrangler 실행내역 패널의 항목 우측 [삭제] 버튼 선택시 해당 실행내역 이후 모든 항목이 삭제됩니다.

위험

Wrangler [완료]를 통해 저장되지 않은 실행 내역들이 포함된 경우 삭제시 해당 설정 항목들은 복구할 수 없습니다. 예) [trim] 삭제시 [trim], [appendString] 항목 삭제
