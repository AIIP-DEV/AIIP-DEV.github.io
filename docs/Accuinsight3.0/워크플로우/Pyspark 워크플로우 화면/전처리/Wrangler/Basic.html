기본 구성

Wrangler 노드는 워크플로우 [데이터 기본처리]에서 제공하는 다양한 노드의 여러 노드를 통합하여 하나의 노드를 구성합니다. Wrangler 노드의 인터페이스를 통해 다양한 데이터 처리의 샘플링된 결과를 실시간으로 확인하고 구성할 수 있습니다. 이 노드에서는 브라우저의 성능에 영향을 줄 수 있으므로 [데이터셋]에 포함된 전체 데이터를 로드하지 않고 일부 데이터를 로드하여 구성하고, 노드 구성을 완료한 후에 전체 데이터에 대해서 전처리를 실행합니다.

노드 생성​

워크플로우 [데이터셋] > [Wrangler] 항목을 선택하여 Wrangler 노드를 생성하고 구성 화면으로 이동합니다.

화면 구성​

노드를 생성하면 선택한 [데이터셋]의 최대 200건의 Row를 로드하고 인터페이스를 구성합니다.

  1. 데이터셋 요약 : [Row], [Column]

  2. 데이터셋 정보 : 로드된 데이터셋/데이터 처리 실시간 확인 패널

    기능설명
    컬럼순서 변경컬럼 상단 [...] 영역 선택 후 Drag&Drop 으로 컬럼 순서를 변경할 수 있습니다.
    컬럽 데이터 포멧 변경컬럼 데이터 타입케스팅
    컬럼명 변경컬럼명을 선택하여 컬럼명을 변경할 수 있습니다.
    분포도 그래프각 컬럼별 데이터 정보가 분포된 Histogram 을 확인할 수 있습니다.

  3. 히스토리 및 처리별 속성 패널 : 실시간으로 처리된 데이터 처리 순으로 각 항목별 요약 정보를 표기합니다. 데이터 처리 중이라면 각 처리 항목에 대한 속성을 편집할 수 있습니다.
