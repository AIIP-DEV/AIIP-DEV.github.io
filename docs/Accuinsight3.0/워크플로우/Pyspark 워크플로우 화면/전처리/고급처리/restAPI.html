restAPI

Definition​

restAPI 호출하여 데이터 로딩 input:이전 노드의 데이터프레임, output:restAPI 호출하는 노드입니다. 우측 Components 패널에서 [전처리] - [고급처리]에서 [restAPI]노드를 선택한 후 항목별 필요정보를 입력합니다. Components 패널의 탭별 항목에서 [더 보기] 버튼을 누르면 입력가능한 전체 항목을 볼 수 있습니다.

Property​

  1. url : restAPI 호출 URL
  2. method : restAPI 요청 방법 ( ex : POST,GET )
  3. request : restAPI 요청 단위, 1회 요청 단위 (default: 1)
  4. header : restAPI 요청 헤더
  5. responseData : 응답 데이터의 위치 e.g. JsonObject 또는 JsonArray 형태 내에 사용할 데이터 path
  6. schema : 응답 데이터의 스키마
  7. 추가 : 입력된 값을 적용하여 데이터 노드를 생성

Example​

  1. Dataset노드를 선택 후 우측 Components 패널에서 [전처리] - [고급처리]에서 노드를 선택하여 워크플로우에 생성
  1. restAPI를 호출할 URL(url) 입력, restAPI 요청(method) POST 혹은 GET 선택, responseData를 입력한 후 schema에서 버튼을 선택하여 응답 데이터의 스키마를 파싱 후 실행
