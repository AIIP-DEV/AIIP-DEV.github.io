Add Timestamp

Timestamp 컬럼을 추가합니다.

워크플로우 보드에서 데이터 노드를 선택하고 노드 메뉴에서 [전처리] 카테고리의 [기본처리]노드중 [Add Timestamp]노드를 선택후 우측 패널에서 항목을 입력합니다.

워크플로우 보드에서 노드를 우클릭하면 메뉴항목을 볼 수 있습니다.

우측 패널​

  1. newColumn : 추가 컬럼명 입력
  2. timestampValue : 날짜값 입력(ex. now-1d , now-1h)
  3. timestampFormat : 날짜 형식 입력(ex. yyyyMMddHHmmss)
  4. 추가 : 입력된 값을 적용하여 데이터 노드를 생성

메뉴​

  • 이름바꾸기 : 작성 중인 노드명 변경
  • 즐겨찾기에 추가 : 즐겨찾기에 [Add Timestamp]노드 추가
  • 삭제 : 해당 노드를 워크플로우에서 제거

예제​

타이타닉 데이터에 하루 전 일자 정보를 "yyyy-MM-dd" 타입 형태로 저장하는 컬럼을 추가하는 워크플로우를 생성합니다.

  1. 워크플로우에 [타이타닉 데이터]를 로드

  2. [타이타닉 데이터] 노드에 [Add Timestamp] 노드를 연결하여 데이터셋을 생성

    • newColumn에 'fixel_date' 입력
    • timestampValue에 'now-1d' 입력
    • timestampFormat에 'yyyy-MM-dd' 입력
    • 추가
