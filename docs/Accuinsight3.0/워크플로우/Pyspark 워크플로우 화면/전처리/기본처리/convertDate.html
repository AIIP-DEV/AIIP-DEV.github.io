Convert Date

timestamp의 날짜포맷을 변경한다.

워크플로우 보드에서 데이터 노드를 선택하고 노드 메뉴에서 [전처리] 카테고리의 [기본처리]노드중 [Convert Date]노드를 선택후 우측 패널에서 항목을 입력합니다.

워크플로우 보드에서 노드를 우클릭하면 메뉴항목을 볼 수 있습니다.

우측 패널​

  1. col : 변경할 컬럼명 입력
  2. newColumn : 추가 컬럼명 입력
  3. oldFormat : 이전 컬럼의 날짜 포맷
  4. newFormat : 신규 컬럼의 날짜 포맷
  5. 추가 : 입력된 값을 적용하여 데이터 노드를 생성

메뉴​

  • 이름바꾸기 : 작성 중인 노드명 변경
  • 즐겨찾기에 추가 : 즐겨찾기에 [Convert Date]노드 추가
  • 삭제 : 해당 노드를 워크플로우에서 제거

예제​

shoping_web 데이터의 timestamp 포맷을 "yyyy-MM-dd HH:mm:ss" 에서 "yyyy-MM-dd"로 변환한 신규 컬럼을 추가하는 워크플로우 생성

  1. 워크플로우에 [shoping_web 데이터]를 로드

  2. [shoping_web 데이터] 노드에 [Convert Date] 노드를 연결하여 데이터셋을 생성

    • col에 'http_date' 선택
    • newColumn에 'new_date' 입력
    • oldFormat에 'yyyy-MM-dd HH:mm:ss' 입력
    • newFormat에 'yyyy-MM-dd' 입력
    • 추가
