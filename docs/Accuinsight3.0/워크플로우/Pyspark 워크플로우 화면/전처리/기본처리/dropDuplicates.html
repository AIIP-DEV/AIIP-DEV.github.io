Drop Duplicates

선택한 컬럼에 중복값이 있으면 제거할 때 사용합니다.

워크플로우 보드에서 데이터 노드를 선택하고 노드 메뉴에서 [전처리] 카테고리의 [기본처리]노드중 [Drop Duplicates]노드를 선택후 우측 패널에서 항목을 입력합니다.

우측 패널의 [더 보기] 버튼을 누르면 입력가능한 전체 항목을 볼 수 있습니다.

워크플로우 보드에서 노드를 우클릭하면 메뉴항목을 볼 수 있습니다.

우측 패널​

  1. subset : 중복값 확인을 위한 컬럼 선택. 적용대상 컬럼이 여러 개일 경우 우측 [+]버튼으로 추가
    • col : 중복값 확인 대상 컬럼
  2. 추가 : 입력된 값을 적용하여 데이터 노드를 생성

메뉴​

  • 이름바꾸기 : 작성 중인 노드명 변경
  • 즐겨찾기에 추가 : 즐겨찾기에 [Drop Duplicates]노드 추가
  • 삭제 : 해당 노드를 워크플로우에서 제거

예제​

타이타닉 데이터에서 sex, age에 중복이 있는 값을 제거 합니다.

  1. 워크플로우에 [타이타닉 데이터]를 로드

  2. [타이타닉 데이터] 노드에 [Select]노드를 연결하여 데이터셋을 생성

    • 적용할 컬럼에 sex, age 컬럼을 선택
    • 추가

  3. [생성된 데이터셋] 노드에 [Sort] 노드를 연결하여 데이터셋을 생성

    • 대상컬럼에 age 컬럼을 선택
    • 정렬기준에 asc 선택
    • 추가

  4. [생성된 데이터셋] 노드에 [Drop Duplicates] 노드를 연결하여 데이터셋을 생성

    • [더 보기]버튼을 클릭하여 입력 항목을 오픈
    • subset의 [+]버튼으로 입력 항목을 추가
    • 중복을 제거할 sex, age 컬럼을 선택
    • 추가
