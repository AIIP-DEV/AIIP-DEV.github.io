Regular Expressions

선택한 컬럼의 문자열에서 정규표현식에 기반하여 매칭되는 문자를 추출하여 새로운 컬럼으로 저장합니다.

워크플로우 보드에서 데이터 노드를 선택하고 노드 메뉴에서 [전처리] 카테고리의 [기본처리]노드중 [Regular Expressions]노드를 선택후 우측 패널에서 항목을 입력합니다.

워크플로우 보드에서 노드를 우클릭하면 메뉴항목을 볼 수 있습니다.

우측 패널​

  1. newColumn : 추가 컬럼명
  2. targetColumn : 추출할 컬럼명
  3. regexp : 정규표현식
  4. group_number : 사용할 표현식의 순서를 입력
  5. 추가 : 입력된 값을 적용하여 데이터 노드를 생성

메뉴​

  • 이름바꾸기 : 작성 중인 노드명 변경
  • 즐겨찾기에 추가 : 즐겨찾기에 [Regular Expressions]노드 추가
  • 삭제 : 해당 노드를 워크플로우에서 제거

예제​

shoping_web 데이터의 user_id에서 정규표현식에 매칭되는 정보로 새로운 컬럼 생성하는 워크플로우를 구성합니다.

  1. 워크플로우에 [shoping_web 데이터]를 로드

  2. [shoping_web 데이터] 노드에 [Regular Expressions] 노드를 연결하여 데이터셋을 생성

    • newColumn에 'two_number' 입력
    • targetColumn에 'user_id' 선택
    • regexp에 '(\d\d)' 입력
    • group_number에 1 입력
    • 추가
