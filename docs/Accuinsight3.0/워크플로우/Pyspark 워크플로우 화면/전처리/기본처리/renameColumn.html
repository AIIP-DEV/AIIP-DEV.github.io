Rename Column

컬럼명을 변경합니다.

워크플로우 보드에서 데이터 노드를 선택하고 노드 메뉴에서 [전처리] 카테고리의 [기본처리]노드중 [Rename Column]노드를 선택후 우측 패널에서 항목을 입력합니다.

워크플로우 보드에서 노드를 우클릭하면 메뉴항목을 볼 수 있습니다.

우측 패널​

  1. columns : 변경할 컬럼과 새로운 컬럼명 입력, 적용대상 컬럼이 여러 개일 경우 우측 [+]버튼으로 추가
    • old Name : 변경 전 컬럼명
    • new Name : 변경 후 컬럼명
  2. 자동입력 : 데이터셋의 모든 컬럼을 자동 입력
  3. 추가 : 입력된 값을 적용하여 데이터 노드를 생성

메뉴​

  • 이름바꾸기 : 작성 중인 노드명 변경
  • 즐겨찾기에 추가 : 즐겨찾기에 [Rename Column]노드 추가
  • 삭제 : 해당 노드를 워크플로우에서 제거

예제​

title이 존재 하지 않는 log 데이터에 컬럼명을 변경하는 워크플로우를 생성

  1. 워크플로우에 [쇼핑몰 로그 데이터]를 로드

    • [쇼핑몰 로그 데이터]노드의 우측 패널에서 부가옵션 설정 클릭
    • format : csv
    • delimeter : ,
    • option : key -> header , value -> false

  2. [쇼핑몰 로그 데이터] 노드에 [Rename Column] 노드를 연결하여 데이터셋을 생성

    • columns 항목을 입력
      • old Name : '_c0' 선택
      • new Name : 'LOG' 입력
    • 추가
