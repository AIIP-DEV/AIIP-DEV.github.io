Drop N/A

결측값이 있는 데이터 삭제시 사용합니다.

워크플로우 보드에서 데이터 노드를 선택하고 노드 메뉴에서 [전처리] 카테고리의 [기본처리]노드중 [Drop N/A]노드를 선택후 우측 패널에서 항목을 입력합니다.

우측 패널의 [더 보기] 버튼을 누르면 입력가능한 전체 항목을 볼 수 있습니다.

워크플로우 보드에서 노드를 우클릭하면 메뉴항목을 볼 수 있습니다.

우측 패널​

  1. drop 처리방법 (method) : any, all, thresh 중 택일
    • any : 대상 columns 중 하나라도 null이 포함된 행을 삭제
    • all : 대상 columns 전체가 null인 행을 삭제
    • thresh : 대상 columns 중 null이 아닌 cell의 갯수가 value보다 적은 행을 삭제
  2. subset : 처리 기준이 되는 대상 columns 설정 (default : 전체 columns)
  3. 추가 : 입력된 값을 적용하여 데이터 노드를 생성

메뉴​

  • 이름바꾸기 : 작성 중인 노드명 변경
  • 즐겨찾기에 추가 : 즐겨찾기에 [Drop N/A]노드 추가
  • 삭제 : 해당 노드를 워크플로우에서 제거

예제​

타이타닉 데이터 중 결측값이 있는 행을 찾아 제거합니다

  1. 워크플로우에 [타이타닉 데이터]를 로드

  1. [타이타닉 데이터] 노드에 [Drop N/A]노드를 연결하여 데이터셋을 생성
    • drop 처리방법을 any로 선택
    • 추가
      • null이 포함된 record 제거

  1. [타이타닉 데이터] 노드에 [Drop N/A]노드를 연결하여 데이터셋을 생성
    • drop 처리방법을 thresh로 선택
    • value 값에 1 입력
    • subset col값에 Age선택
    • 추가
      • Age컬럼에 null이 한개 이상 있는 record 제거
