Substring

문자열을 잘라서 새로운 컬럼으로 추가합니다.

워크플로우 보드에서 데이터 노드를 선택하고 노드 메뉴에서 [전처리] 카테고리의 [기본처리]노드중 [Substring]노드를 선택후 우측 패널에서 항목을 입력합니다.

워크플로우 보드에서 노드를 우클릭하면 메뉴항목을 볼 수 있습니다.

우측 패널​

  1. subset : col, startPosition, length, newColumn 정보 입력. 적용대상 컬럼이 여러 개일 경우 우측 [+]버튼으로 추가
    • col : 기준 컬럼 선택
    • startPosition : 시작 위치 입력
    • length : 길이 입력
    • newColumn : 신규 컬럼 명 입력
  2. 추가 : 입력된 값을 적용하여 데이터 노드를 생성

메뉴​

  • 이름바꾸기 : 작성 중인 노드명 변경
  • 즐겨찾기에 추가 : 즐겨찾기에 [Substring]노드 추가
  • 삭제 : 해당 노드를 워크플로우에서 제거

예제​

타이타닉 데이터에서 name 컬럼의 처음 5글자를 새로운 컬럼으로 만듭니다.

  1. 워크플로우에 [타이타닉 데이터]를 로드

  2. [타이타닉 데이터] 노드에 [Substring] 노드를 연결하여 데이터셋을 생성

    • 적용할 컬럼에 Name 컬럼을 선택
    • startPosition에 1 입력
    • length에 5 입력
    • newColumn에 'name_5' 입력
    • 추가
