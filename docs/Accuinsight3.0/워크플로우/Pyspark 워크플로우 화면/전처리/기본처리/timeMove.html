Time Move

timestamp 타입의 컬럼 시간을 설정된 값만큼 이동합니다.

워크플로우 보드에서 데이터 노드를 선택하고 노드 메뉴에서 [전처리] 카테고리의 [기본처리]노드중 [Time Move]노드를 선택후 우측 패널에서 항목을 입력합니다.

워크플로우 보드에서 노드를 우클릭하면 메뉴항목을 볼 수 있습니다.

우측 패널​

  1. targetColumn : 목록 중 기준 컬럼 선택(선택 컬럼은 timestamp data type이나 이 timestamp data type이 들어있는 문자열이여야 함)
  2. targetColumnFormat : 날짜 포맷을 지정
  3. newColumn : 신규 컬럼 명
  4. time : 이동할 시간(ex. 3600(1시간 후), -3600(1시간 전)
  5. 추가 : 입력된 값을 적용하여 데이터 노드를 생성

메뉴​

  • 이름바꾸기 : 작성 중인 노드명 변경
  • 즐겨찾기에 추가 : 즐겨찾기에 [Time Move]노드 추가
  • 삭제 : 해당 노드를 워크플로우에서 제거

예제​

timestamp 타입의 컬럼을 10시간 후의 데이터를 갖는 새로운 컬럼을 추가하는 워크플로우를 구성합니다.

  1. 워크플로우에 [shoping_web 데이터]를 로드

  2. [shoping_web 데이터] 노드에 [Time Move] 노드를 연결하여 데이터셋을 생성

    • targetColumn에 'http_date' 컬럼 선택
    • targetColumnFormat에 'yyyy-MM-dd HH:mm:ss' 입력
    • newColumn에 'new_date' 입력
    • time에 36000 입력
    • 추가
