Replace

특정 컬럼의 값을 지정한 값으로 치환합니다.

워크플로우 보드에서 데이터 노드를 선택하고 노드 메뉴에서 [전처리] 카테고리의 [기본처리]노드중 [Replace]노드를 선택후 우측 패널에서 항목을 입력합니다.

우측 패널의 [더 보기] 버튼을 누르면 입력가능한 전체 항목을 볼 수 있습니다.

워크플로우 보드에서 노드를 우클릭하면 메뉴항목을 볼 수 있습니다.

우측 패널​

  1. checkAllMatch: 전체 문자열이 일치할 경우에만 치환하기
  2. to_replace : 변경전 값 입력(문자열 입력시 쌍따옴표("") 사용필요)
  3. value : 대치값(변경후 값) 입력(문자열 입력시 쌍따옴표("") 사용필요)
  4. subset : replace 적용 컬럼선택. 전체 컬럼 대상으로 할 경우 생략
  5. 추가 : 입력된 값을 적용하여 데이터 노드를 생성

메뉴​

  • 이름바꾸기 : 작성 중인 노드명 변경
  • 즐겨찾기에 추가 : 즐겨찾기에 [Replace]노드 추가
  • 삭제 : 해당 노드를 워크플로우에서 제거

예제​

타이타닉 데이터 중 sex컬럼의 값을 변경 합니다. ("male" > "남성")

  1. 워크플로우에 [타이타닉 데이터]를 로드

  2. [타이타닉 데이터] 노드에 [Replace] 노드를 연결하여 데이터셋을 생성

    • to_replace 값에 "male" 입력
    • value 값에 "남성" 입력
    • [더 보기]버튼을 클릭하여 입력 항목을 오픈
    • subset의 [+]버튼으로 입력 항목을 추가
    • 적용할 sex 컬럼을 선택
    • 추가
