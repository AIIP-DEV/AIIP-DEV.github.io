Add Condition Column

조건식을 입력하여 True/False에 매핑되는 값을 새로운 컬럼에 추가합니다.

워크플로우 보드에서 데이터 노드를 선택하고 노드 메뉴에서 [전처리] 카테고리의 [기본처리]노드중 [Add Condition Column]노드를 선택후 우측 패널에서 항목을 입력합니다.

워크플로우 보드에서 노드를 우클릭하면 메뉴항목을 볼 수 있습니다.

우측 패널​

  1. expression : 사용자가 원하는 조건식 직접 입력(띄어쓰기 필수). 적용할 조건식이 여러 개 일 경우 오른쪽 [+]버튼 선택
    • customCondition : 조건식
    • trueValue : 조건식이 True일 경우 매핑 될 값
    • falseValue : 조건식이 False일 경우 매핑 될 값
    • newColumn : expression 결과가 입력될 새로운 컬럼(미입력시 조건식을 적절하게 지환[필드명조건값]하여 지정)
  2. 추가 : 입력된 값을 적용하여 데이터 노드를 생성

메뉴​

  • 이름바꾸기 : 작성 중인 노드명 변경
  • 즐겨찾기에 추가 : 즐겨찾기에 [Add Condition Column]노드 추가
  • 삭제 : 해당 노드를 워크플로우에서 제거

예제​

타이타닉 데이터에 young_man 컬럼을 추가하여 Age컬럼 값이 30보다 작고, Sex컬럼 값이 'male'인 레코드에 매핑값(2) 그외의 레코드에 매핑값(1)을 입력하는 워크플로우 생성

  1. 워크플로우에 [타이타닉 데이터]를 로드

  2. [타이타닉 데이터] 노드에 [Add Condition Column] 노드를 연결하여 데이터셋을 생성

    • customCondition에 [Age < 30 and Sex='male'] 입력
    • trueValue에 2 입력
    • falseValue에 1 입력
    • newColumn에 'young_man' 입력
    • 추가
