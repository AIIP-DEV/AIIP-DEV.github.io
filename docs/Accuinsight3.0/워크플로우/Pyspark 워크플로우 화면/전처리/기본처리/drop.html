Drop

선택한 컬럼을 삭제하는 노드입니다.

워크플로우 보드에서 데이터 노드를 선택하고 노드 메뉴에서 [전처리] 카테고리의 [기본처리]노드중 [Drop]노드를 선택후 우측 패널에서 항목을 입력합니다.

워크플로우 보드에서 노드를 우클릭하면 메뉴항목을 볼 수 있습니다.

우측 패널​

  1. col : 삭제할 컬럼 선택
  2. 추가 : 입력된 값을 적용하여 데이터 노드를 생성

메뉴​

  • 이름바꾸기 : 작성 중인 노드명 변경
  • 즐겨찾기에 추가 : 즐겨찾기에 [Drop]노드 추가
  • 삭제 : 해당 노드를 워크플로우에서 제거

예제​

타이타닉 데이터에서 Survived, Embarked, Cabin, Fare, Ticket, Parch, SibSp 컬럼을 제거하는 워크플로우를 작성 합니다.

  1. 워크플로우에 [타이타닉 데이터]를 로드

  2. [타이타닉 데이터] 노드에 [Drop]노드를 연결하여 데이터셋을 생성

    • 삭제할 컬럼에 Survived, Embarked, Cabin, Fare, Ticket, Parch, SibSp 컬럼을 선택
    • 추가
