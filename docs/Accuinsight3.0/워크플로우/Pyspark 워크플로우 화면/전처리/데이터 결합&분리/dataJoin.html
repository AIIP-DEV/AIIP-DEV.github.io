Data Join

2개 이상의 노드로부터 데이터를 Join하는 노드입니다.

워크플로우 보드에서 데이터 노드를 선택하고 노드 메뉴에서 [전처리] 카테고리의 [데이터 결합/분리]노드중 [Data Join]노드를 선택후 우측 패널에서 항목을 입력합니다.

우측 패널​

  1. Suggestion: 결합 대상 추천 팝업
  2. columns : 결합될 대상 컬럼들
    • col1 : 첫 번째 노드 기준 컬럼 선택
    • col2 : 두 번째 노드 기준 컬럼 선택
    • condition: col1과 col2의 결합 조건
  3. how : join 유형 선택
    • join유형 설명
      • inner : 조인 조건에 맞는 행만 검색
      • outer : 조인 키에 대응되는 row가 없는 경우 null로 추출
      • left_outer : col1에서 선택한 조인 키에 대응되는 row가 없는 경우 null로 추출
      • right_outer : col2에서 선택한 조인 키에 대응되는 row가 없는 경우 null로 추출
      • left_semi : col1과 col2가 서로 join시 col1,col2의 값이 같은 데이터1추출 (exists 함수와 유사)
      • left_anti : col1과 col2가 서로 join시 col1,col2의 값이 없는 데이터1추출 (not exists 함수와 유사)
      • cross : 두 노드의 카티션 프로덕트(곱집합)결과를 추출
  4. cleanColumns : col2 컬럼 제거 (중복되는 key Column)

Suggestion​

  1. 결과 내 검색
  2. 추천 리스트
    • radio : 적용할 결과 선택
    • column1 : 첫번째 데이터셋의 컬럼
    • column2 : 두번째 데이터셋의 컬럼
    • similarity : 연결 유사성 비율
