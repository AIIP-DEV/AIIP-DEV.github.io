Split Column

구분자를 통해 컬럼을 분할 하여 새로운 컬럼을 생성 합니다.

워크플로우 보드에서 데이터 노드를 선택하고 노드 메뉴에서 [전처리] 카테고리의 [데이터 결합/분리]노드중 [Split Column]노드를 선택후 우측 패널에서 항목을 입력합니다.

우측 패널의 [더 보기] 버튼을 누르면 입력가능한 전체 항목을 볼 수 있습니다.

워크플로우 보드에서 노드를 우클릭하면 메뉴항목을 볼 수 있습니다.

우측 패널​

  1. column : 대상 컬럼
  2. delimiter : 분할할 구분자
  3. newColumns : 분할하여 생성될 컬럼명, 분할되는 컬럼이 여러 개일 경우 우측 [+]버튼으로 추가 (순서있음)
    • column : 컬럼명
  4. 추가 : 입력된 값을 적용하여 데이터 노드를 생성

메뉴​

  • 이름바꾸기 : 작성 중인 노드명 변경
  • 즐겨찾기에 추가 : 즐겨찾기에 [Split Column]노드 추가
  • 삭제 : 해당 노드를 워크플로우에서 제거

예제​

[shoping_web] 데이터의 request 컬럼을 request_url 컬럼과 request_query 컬럼으로 분할 하는 워크플로우를 작성 합니다.

  1. 워크플로우에 [shoping_web] 데이터를 로드

  2. [shoping_web] 노드에 [Split Column]노드를 연결하여 데이터셋을 생성

    • column 항목을 'request' 컬럼 선택
    • delimiter 항목에 '\\?' 입력
    • newColumns 항목에서 [+]버튼으로 두개의 입력을 생성
      • 첫번째 column 항목에 'request_url' 입력
      • 두번째 column 항목에 'request_query' 입력
    • 추가
