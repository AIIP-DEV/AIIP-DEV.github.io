ssh

Definition​

원격 host에 ssh 명령어 실행시 사용합니다. 우측 Components 패널에서 [작업]에서 [ssh]노드를 선택한 후 Property 항목을 입력합니다. Components 패널의 탭별 항목에서 [더 보기] 버튼을 누르면 입력가능한 전체 항목을 볼 수 있습니다.

Property​

  1. IP : SSH HOST 주소
  2. PORT : SSH port
  3. 사용자명 : SSH 사용자명
  4. password : 비밀번호
  • password : ssh 사용자 비밀번호
  • pemFile : ssh 연결할 pem 파일 선택
  1. command : SSH에서 실행할 commnad 입력
  2. argument : SSH에서 사용할 argument

Example​

  1. 우측 Components 패널에서 [작업]에서 노드를 선택하여 워크플로우에 생성
  1. IP, PORT, username, password입력, command에서 버튼을 선택하여 editor에 shell command를 작성, argument는 추가 버튼을 선택하여 shell에서 사용할 인수 입력 후 전체 워크플로우 실행
