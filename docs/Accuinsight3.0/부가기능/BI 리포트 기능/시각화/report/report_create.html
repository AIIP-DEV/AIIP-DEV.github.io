리포트 작성

워크플로우에서 생성한 차트에서 리포트 작성 방법에 대하여 설명합니다.

차트 추가​

오른쪽 차트 목록을 드래그 & 드랍하여 리포트에 차트를 추가 할 수 있습니다.

  1. 리포트에 추가된 차트의 화면 크기를 조절합니다.
  2. 차트를 리포트에서 제거합니다.
    • 리포트에서 단순히 차트를 제거하는 것으로, 차트를 삭제하는 동작은 아닙니다.
  3. 리포트에 포함된 차트가 수정되지 않도록 합니다.
  4. 차트를 수정합니다.
    • 차트 생성화면으로 이동하여 차트를 수정할 수 있습니다.
  5. 차트에 필터를 적용합니다.

오브젝트 추가​

오른쪽 오브젝트를 드래그 & 드랍하여 리포트에 차트를 추가 할 수 있습니다.

텍스트 오브젝트 추가​

  1. 리포트에 추가된 오브젝트의 화면 크기를 조절합니다.
  2. 오브젝트를 리포트에서 제거합니다.
  3. 오브젝트를 수정합니다.

전역 차트 데이터 필터 추가​

  1. 오브젝트 항목에서 필터 추가 아이콘을 선택합니다.
  2. 추가된 필터 이름과 필터 연산자를 보여줍니다.
  3. 필터 수정 화면으로 진입합니다.
  4. 필터를 삭제합니다.

필터 수정 화면입니다.

  1. 사용자가 2번 선택된 컬럼에 대한 별칭을 부여할 할 수 있습니다.
  2. 데이터에 포함된 컬럼명입니다.
    • 리포트에 포함된 차트들의 공통 컬럼만 표시됩니다.
  3. 필터에 적용할 데이터 형식을 선택합니다.
    • text input: 텍스트 형식의 입력입니다.
    • number input: 숫자 형식의 입력입니다.
    • select: 선택된 컬럼의 값들을 입력할 수 있습니다.
    • datapicker: 날짜를 선택합니다.
    • range number input: 숫자 범위를 선택합니다.
    • range datepicker: 날짜 범위를 선택합니다.
  4. 필터에 대한 연산자를 선택합니다.
    • <: less than: 입력값보다 작은 값을 반환합니다.
    • <=: less than or equal: 입력값 이하의 값을 반환합니다.
    • =: equal: 입력값과 같은 값을 반환홥니다.
    • !=: not equal: 입력값과 같지 않은 값을 반환합니다.
    • >=: greater than or equal: 입력값과 같거나 큰 값을 반환합니다.
    • >: greater than: 입력값과 큰 값을 반환합니다.
    • %A%: contains: 입력값에 포함된 값을 반환합니다.
    • ~: between: 지정된 범위내의 값을 반환합니다.
  5. 입력 엘리먼트에서 select가 선택된 경우에 활성화 됩니다.
    • 선택된 컬럼에 실제 존재하는 값을 입력해야 합니다.
  6. 필터를 삭제합니다.
  7. 다른 필터를 추가합니다.

Report에 차트 추가​

Report에 차트를 추가하는 방법과 차트를 수정으로부터 보호하는 기능에 대한 방법입니다.

  • 동일한 Area 차트 2개를 추가합니다.
    • 우측의 차트 목록에서 Area 차트를 드래그 & 드랍합니다.
  • 첫번째 Area 차트를 수정으로부터 보호합니다.
    • 자물쇠 아이콘을 클락하여 잠금 아이콘으로 변경합니다.
  • 두번째 Area 차트를 수정 합니다.
    • 연필 모양의 편집 아이콘을 클릭합니다.
  • 수정 후 두번째 Area 차트에만 변경이 적용됩니다.

특정 차트에 적용될 필터 추가​

특정 차트에만 필터를 추가하는 기능에 대한 설명입니다.

  • 기본적인 동작은 전역 차트 데이터 필터 추가와 동일합니다.
  • 전역 차트 데이터 필터 추가를 통해 추가된 차트가 있을 경우 모든 필터가 함께 적용됩니다.

Report에 오브젝트 추가​

Report에 오브젝트를 추가하는 방법에 대한 방법입니다.

  • 텍스트 타입의 오브젝트를 추가하는 예시입니다.

  • 필터 타입의 오브젝트를 추가하는 예시입니다.
    • 적용되는 필터에 따라 차트가 변경되는 것을 확인할 수 있습니다.

Report 다운로드​

Report를 다운로드하는 기능에 대한 설명입니다.

  1. 리포트를 이미지로 다운로드 합니다.

  2. 리포트를 HTML로 다운로드 합니다.

    1. 다운로드된 HTML의 형식 예시입니다.

      <!DOCTYPE html>
      <html lang="en">
      <head>
          <meta charset="UTF-8">
          <meta name="viewport" content="width=device-width, initial-scale=1.0">
          <title>Report-todo</title>

      </head>

      <body style="margin:0px;padding:0px;overflow:scroll">
          <div>
              <iframe src="http://aiip-dev.skcc.com/reportview?projectId=2329&workflowId=11281&reportId=1965&accessToken=<Enter your access token>" frameborder="0" style="overflow:scroll;height:960px;width:100%;position:absolute;top:0px;left:0px;right:0px;bottom:0px" height="100%" width="100%"></iframe>
          </div>
      </body>

      </html>

주의
  • 위 HTML 파일의 accessToken=<Enter your access token> 부분을 발급 받은 토큰으로 변경하여야 합니다.
  • 토큰 발급 화면으로 이동합니다.
    • 발급된 토큰을 <Enter your access token> 부분에 복사하여 줍니다.
    • 토큰 발급 절차는 아래와 같습니다.
      • 서비스는 통합 선택
      • API는 리포트 다운로드 선택
      • 범위는 다운로드 받고자 하는 리포트 이름 중에서 선택
  • 발급된 토큰의 사용 예시입니다.
  • ex) accessToken=eyJ0eXAiOiJKV1QiLCJhbGc... :::

다운로드 받은 HTML 파일을 인터넷 브라우저로 끌어 넣으면 리포트가 표출됩니다. :::
