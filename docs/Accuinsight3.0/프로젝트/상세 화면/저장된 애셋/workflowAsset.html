워크플로우 애셋

워크플로우 애셋 개요​

  • 프로젝트에 저장된 워크플로우 애셋을 조회/삭제/새 워크플로우 시작을 위해 관리합니다.
​

워크플로우 애셋 화면​

  1. 워크플로우 애셋 목록을 열거나 닫음
  2. 파일 형태의 백업본을 로드하여 애셋을 등록할 수 있습니다.
  3. 워크플로우 애셋의 More Action 메뉴
    • 새 워크플로우 시작 - 워크플로우 애셋을 사용하여 새로운 워크플로우를 생성 합니다.
    • 백업 - 애셋을 파일 형태로 로컬 디스크에 저장합니다.
    • 삭제 - 워크플로우 애셋을 삭제 합니다. 삭제 권한이 있을 때에만 표시 됩니다.
  4. 페이지 선택
  5. 페이지당 표시할 애셋 개수
​

1. 보기 모드​

카드 뷰와 리스트 뷰를 선택할 수 있습니다.

  • 워크플로우 애셋 카드

  • 워크플로우 애셋 리스트

  • 애셋 구성요소 표시

    • : 워크플로우 애셋에 리포트가 있음
    • : 워크플로우 애셋에 샌드박스가 있음
    • : 워크플로우 애셋에 모델이 있음
​

2. More Actions​

  • 워크플로우 애셋 카드 More Action
    카드에 마우스 커서를 올리면 More Action 버튼이 표시되며, 워크플로우 애셋은 다음과 같은 메뉴를 제공합니다.

    • 새 워크플로우 시작: 워크플로우 애셋을 사용하여 새로운 워크플로우를 생성 합니다.
    • 백업: 애셋을 파일 형태로 로컬 디스크에 저장합니다.
    • 삭제: 워크플로우 애셋을 삭제 합니다. 삭제 권한이 있을 때에만 표시 됩니다.

  • 워크플로우 애셋 리스트 More Action More Action 버튼은 항상 표시되며, 카드 뷰와 동일한 메뉴를 제공 합니다.

​

다중 선택 More Actions​

여러개의 항목을 선택해서 More Actions를 실행할 수 있는 기능을 제공 합니다. 워크플로우를 생성할 때에는 하나의 워크플로우 애셋만 사용 가능하기 때문에, 선택한 항목 개수에 따라 표시되는 액션 버튼이 달라집니다.

  • 카드 다중 선택
    한개만 선택 했을 때에는 삭제와 워크플로우 시작, 복원 버튼이 표시 됩니다.
    두개 이상 선택 했을 때에는 삭제, 복원 버튼만 표시 됩니다.
  • 리스트 다중 선택
    한개만 선택 했을 때에는 삭제와 워크플로우 시작, 복원 버튼이 표시 됩니다.

두개 이상 선택 했을 때에는 삭제, 복원 버튼만 표시 됩니다.

​

워크플로우 애셋 기능 소개​

​

새 워크플로우 시작​

선택한 애셋에 저장된 구성요소로 새 워크플로우를 생성합니다. 상단에 위치한 '새 워크플로우 시작'버튼을 선택하면 프로젝트 선택 팝업이 표시 됩니다.

워크플로우 애셋 복사를 진행할 프로젝트를 선택하고 등록 버튼을 눌러 진행합니다.

워크플로우 애셋 복사 완료 후 선택한 프로젝트에 워크플로우가 생성됩니다.
