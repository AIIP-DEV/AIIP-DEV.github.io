스케줄러 목록

  • 워크플로우 스케줄링에 필요한 기능들을 제공합니다.

​

1. 헤더​

  1. Chevron - 이전 화면으로 이동
​

2. 탐색​

  1. 보기 모드 - 리스트 보기 모드와 캘린더 모드에 대해서 토글 수행
  2. 검색창 - 스케줄러의 이름으로 검색 수행
  3. 생성자 필터 - 스케줄러를 생성한 사용자로 검색 수행
  4. 워크플로우 필터 - 스케줄러가 속한 워크플로우로 검색 수행. 기본 설정값은 프로젝트 내 스케줄러가 생성된 전체 워크플로우로 지정
  5. 활성 상태 필터 - 체크 박스 선택 시 활성 상태 Active로 검색 수행
  6. 소팅 필터 - 최신 실행 순, 최신 업데이트 순, 제목 오름차순, 제목 내림차순으로 소팅 진행
  7. 스케줄러 등록 - 워크플로우 선택 팝업 표시 후 스케줄러를 등록할 워크플로우를 선택하면 상세 정보 입력 화면으로 이동
​

3. 스케줄러 목록​

  1. 항목 선택 - 스케줄러 항목을 선택/해제
  2. 활성상태 - 스케줄러를 Active/Inactive로 설정
  3. 스케줄러의 More Actions 메뉴
    • 편집 - 스케줄러 편집 화면으로 이동
    • 복제 - 스케줄러 복제 수행
    • 삭제 - 스케줄러 삭제 수행
  4. 페이지 설정
  5. 한 페이지에 표시할 항목 수 설정
​

4. 캘린더 모드​

  • 월별 표시 - 일 단위로 스케줄러 동작 현황을 표시
  1. 확인하고자 하는 월로 이동
  2. 오늘 버튼 클릭 시 오늘에 해당하는 월로 이동
  3. 월, 일 토글 버튼 - radio 버튼을 월로 체크 시 캘린더 모드의 월별 표시로 전환, 일로 체크 시 캘린더 모드의 일별 표시로 전환
  4. 스케줄러 등록 - 워크플로우 선택 팝업 표시 후 스케줄러를 등록할 워크플로우를 선택하면 상세 정보 입력 화면으로 이동
  • 일별 표시 - 시간 단위로 스케줄러 동작 현황을 표시
  1. 오늘 버튼 클릭 시 오늘에 해당하는 일자의 스케줄러 동작 현황 표시
  2. 월, 일 토글 버튼 - radio 버튼을 월로 체크 시 캘린더 모드의 월별 표시로 전환, 일로 체크 시 캘린더 모드의 일별 표시로 전환
  3. 스케줄러 등록 - 워크플로우 선택 팝업 표시 후 스케줄러를 등록할 워크플로우를 선택하면 상세 정보 입력 화면으로 이동
  4. 달력에서 원하는 일자 선택 시 해당 날짜의 스케줄러 동작 현황 표시
  5. 선택한 날짜의 요일과 일자 표시
  6. 현재 시간을 표시
