스케줄러 상세

  • 스케줄러 진행 상태를 확인할 수 있고, 스케줄러 설정 정보를 수정할 수 있습니다.
​

스케줄러 정보​

​

1. 헤더​

  1. Chevron - 이전 화면으로 이동
  2. 수정 - 스케줄 설정 수정
  3. More Action - 삭제 버튼, 스케줄이 inactive 상태일때 삭제 가능
​

2. 탭​

  1. 스케줄러 정보 탭 - 스케줄러 등록 정보 표시
  2. 진행상태 탭 - 진행상태 표시
​

3. 기본 정보​

  1. 스케줄러 이름 확인
  2. 실행 이력 보관주기 확인
​

4. 실행주기​

스케줄 실행 주기 설정을 확인할 수 있습니다.

​

5. 알림​

스케줄 알람 설정을 확인할 수 있습니다.

정보

서비스 관리자가 Admin 콘솔에서 알림 관리에 등록한 내용이 알림 채널에 표시 됩니다.

​

6. 변수​

스케줄 변수 설정을 확인할 수 있습니다.

정보

프로젝트 변수는 선택만 가능하고 수정이 불가능 합니다.
수정이 필요할 경우 변수 그룹 관리에서 수정을 해야 합니다.

​
​

진행 상태​

  1. Chevron - 이전 화면으로 이동
  2. 스케줄러 정보 탭 - 스케줄러 등록 정보 표시
  3. 진행상태 탭 - 진행상태 표시
  4. Job ID - 회차 마우스 오버 시 Job ID 표시
  5. 상세 정보 - 각 회차 클릭 시 Job 상세정보 팝업으로 표시
  6. 특정 Job 재실행 - 완료나 실패 상태의 Job을 재실행하고자 하는 경우 box의 재실행 버튼 클릭

스케줄 Job 재실행 팝업​

  1. 선택한 Job의 정보
  2. 워크플로우 리스트 - 수정하고자 하는 변수의 소속 워크플로우 선택 (워크플로우 / 서브워크플로우)
  3. 2.에서 선택한 워크플로우의 변수 리스트 - 재실행 시 수정할 변수의 값을 수정
  4. 시간 변수 처리 기준 - #{now} 시간변수를 사용한 경우 재실행 시 현재 일자가 들어가게 할 것인지, Job의 이전 시작일자(1.에서 확인 가능한 '시작' 일자)가 들어가게 할 것인지 선택
  5. 재실행 - 클릭 시 재실행되며 진행상태에 신규 회차로 확인 가능

상세정보 팝업​

  1. Spark - Spark Log
  2. Airflow - Airflow Log
  3. 변수 - 스케줄에 설정된 변수
