프로젝트 애셋 백업/복구

프로젝트에 저장된 생성한 워크플로우, 데이터셋 애셋을 .zip파일로 저장하거나 백업한 .zip파일로 프로젝트를 복원하여 확인이 가능합니다.

워크플로우 애셋 백업​

프로젝트의 저장된 애셋 화면에서 워크플로우 애셋 목록에서 백업할 애셋의 MoreAction 버튼 선택 후 나오는 백업 메뉴를 선택하면 .zip형식으로 백업파일을 다운로드합니다.

정보

다운로드 받는 파일은 "Backup_WorkflowAsset_{워크플로우애셋ID}.zip" 형태로 다운로드 받습니다.

워크플로우 애셋 복구​

탐색영역의 "복원" 버튼을 누른 후 복구할 워크플로우 애셋 백업파일(Backup_WorkflowAsset_{워크플로우애셋ID}.zip)을 선택하여 "열기"를 진행하면 선택한 워크플로우 애셋이 목록에 복원됩니다.

데이터셋 애셋 백업​

프로젝트의 저장된 애셋 화면에서 데이터셋 애셋 목록에서 백업할 애셋의 MoreAction 버튼 선택 후 나오는 백업 메뉴를 선택하면 .zip형식으로 백업파일을 다운로드합니다.

정보

다운로드 받는 파일은 "Backup_DatasetAsset_{데이터셋애셋ID}.zip" 형태로 다운로드 받습니다.

데이터셋 애셋 복구​

탐색영역의 "복원" 버튼을 누른 후 복구할 데이터셋 애셋 백업파일(Backup_DatasetAsset_{데이터셋애셋ID}.zip)을 선택하여 "열기"를 진행하면 선택한 데이터셋 애셋이 목록에 복원됩니다.
