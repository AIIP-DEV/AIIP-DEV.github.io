워크플로우

제공하는 다양한 노드를 활용하여 워크플로우를 구성합니다. 우측에 있는 노드 메뉴와 우측 패널을 통해 노드를 선택하여 연결하고, 노드에서 제공하는 기능으로 데이터, 전처리, 모델링, 하둡작업 등을 수행하여 최종 데이터셋을 만드는 것이 일반적인 작업패턴입니다.
또는 제공되는 템플릿이나 저장된 애셋을 이용한 워크플로우 구성도 가능합니다.

워크플로우 화면 구성​

  1. 워크플로우 바
  2. 워크플로우 보드
  3. 우측 패널
  4. 노드 메뉴
  5. 노드

1. 워크플로우 바​

  1. 워크플로우명
    • 워크플로우명을 보여줍니다. 마우스 hover 시 워크플로우명을 편집할 수 있는 아이콘이 나타납니다.
  2. 저장
    • 워크플로우를 저장합니다.
  3. More Actions
    • 전체 워크플로우 실행
      • 전체 워크플로우를 실행합니다.
    • 다른 이름으로 저장
      • 워크플로우를 다른 이름으로 저장합니다.
    • 워크플로우 애셋 저장
      • 워크플로우를 애셋으로 저장합니다.
    • 이름 바꾸기
      • 워크플로우 이름을 변경합니다.
    • 스케줄러 설정
      • 워크플로우 스케줄러를 설정합니다.
        • 기본 정보 설정
          • 스케줄러 이름
          • 실행 이력 보관주기 선택
        • 스케줄 설정
          • 기본 방식, Cron 표현식
          • 시작일시, 반복 주기, 종료 일시
        • 알림 설정
          • 알림 대상 이벤트
          • 알림 채널
        • 변수 추가
          • 워크플로우 변수
    • 워크플로우 로그
      • 워크플로우 로그를 조회합니다.
    • 전체 실행내역
      • 워크플로우 전체(스케줄 포함) 실행 이력을 조회합니다.
      • 로그 조회 버튼을 통해 Job 로그(Spark, Airflow)를 확인합니다.
        • Spark - Spark Log
        • Airflow - Airflow Log
        • 변수 - 워크플로우 설정 변수
    • 워크플로우 변수
      • 워크플로우 변수를 추가, 편집, 삭제합니다.
    • Livy 서버 설정
      • Livy 서버을 설정합니다.
    • Spark 자원 설정
      • Spark 자원을 설정합니다.
        • Driver Memory
        • Executor Memory
      • Configuration을 설정합니다.
        • Configuration key와 value를 입력해 설정합니다.
    • 삭제
      • 워크플로우를 삭제합니다.

2. 워크플로우 보드​

워크플로우 생성/편집을 위한 기본화면입니다.

  1. 워크플로우 보드에서 마우스 우클릭하면 워크플로우 바의 More Actions와 같은 워크플로우 옵션 메뉴가 나타납니다.
  2. 좌측 하단의 배율 버튼, 혹은 ctrl + 마우스 휠 조작을 통해 워크플로우 보드를 확대, 축소할 수 있습니다.

3. 우측 패널​

노드 메뉴 및 노드 선택에 맞는 정보를 보여주는 영역입니다.

4. 노드 메뉴​

  • 즐겨찾기, 데이터, 전처리, 모델링, 하둡작업으로 구성되어 있습니다.
  • 노드 메뉴를 선택하면 우측 패널에 포함된 노드들의 목록이 표시됩니다.

5. 노드​

워크플로우 보드에 워크플로우의 노드들이 표시되어집니다.
워크플로우 보드에서 노드를 선택한 후 우측 패널에 있는 노드를 선택하면 노드를 추가하여 워크플로우를 구성할 수 있습니다.

노드 상태​

노드는 상태에 따라 노드 아이콘이 구분되어집니다.

  • 입력, 실행전
    • 새 노드를 추가했을 때의 초기 상태입니다.
  • 진행중
    • 해당 노드 작업이 실행중일 때의 상태입니다.
  • 완료
    • 해당 노드 작업이 완료되었을 때의 상태입니다.
  • hover
    • 노드 위에 마우스 커서가 위치했을 때의 상태입니다.
  • selected
    • 노드를 선택한 상태입니다.

노드 상태 아이콘​

노드는 상태 변화에 따라 노드 아이콘의 우측 상단에 상태 아이콘이 나타납니다.

  • 리더보드
    • 리더보드를 포함하고 있는 노드에 표시되는 아이콘입니다.
  • 진행중
    • 해당 노드 작업이 실행중일 때 표시되는 아이콘입니다.
  • 완료
    • 해당 노드 작업이 완료되었을 때 표시되는 아이콘입니다.
  • 에러
    • 해당 노드 작업이 실패하였을 때 표시되는 아이콘입니다.

노드 옵션​

노드를 선택하고 마우스 우클릭을하면 노드별로 사용 가능한 노드 옵션 메뉴가 나타납니다 (노드별 옵션 메뉴는 워크플로우 화면의 노드별 설명에서 확인할 수 있습니다).

실행중인 spark 노드의 UI 접속​

spark작업을 수행하는 노드의 경우 우클릭 시 spark UI 접속 기능을 제공합니다.

  • 전체 워크플로우 실행이나 스케줄을 통한 워크플로우 실행 시에만 제공됩니다.
  • 노드 작업이 완료된 경우에는 제공되지 않습니다. (실행중인 경우에만 접속 가능)

  • 대상 노드

    • 데이터, 전처리 노드 전체 / pyspark 노드
    • 위의 노드들로 구성된 워크플로우를 참조하는 subWorkflow 노드

  • UI가 구성중인 경우 Spark UI 접속 클릭 시 아래와 같이 alert을 확인할 수 있습니다.

  • UI가 활성화되면 새창으로 접속됩니다.
