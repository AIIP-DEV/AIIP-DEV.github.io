API 키 관리

외부에서 API를 호출하기 위한 Key를 생성하고 관리 수행합니다.

API 키 관리 화면​

1. API Key 검색​

  • 입력한 키워드가 설명 혹은 이름과 일치하는 API Key를 목록에 표시합니다.

2. API Key 목록​

  • 전체 선택 - 체크시 모든 항목을 선택합니다.
  • 목록 클릭 시 API 키 관리 화면으로 이동합니다.
  • 이름
    • Key 등록 시 설정한 이름을 표시합니다.
  • 설명
    • Key 등록 시 설정한 설명을 표시합니다.
  • ID
    • 발급된 Key ID를 표시합니다.
    • 클립보드 아이콘을 선택하여 해당 값을 복사할 수 있습니다.
  • Key
    • 발급된 API Key를 표시합니다.
    • 클립보드 아이콘을 선택하여 해당 값을 복사할 수 있습니다.
  • 생성자
    • Key 생성 계정을 표시합니다.
  • 생성일
    • Key 작성일시를 표시합니다.
  • More Actions 버튼
    • 편집
    • Key 복사
    • Token 발급
    • API 목록
    • Key 재발급
    • 삭제

3. 발급​

  • 새로운 API Key를 발급합니다.
  • 이름
    • Key 이름을 입력합니다.
    • 필수 항목
  • 설명
    • Key 설명을 입력합니다.
    • 필수 항목
  • 범위 설정
    • 서비스 - 대상 서비스를 지정합니다.
    • API - 선택한 서비스에서 이용가능한 API를 지정합니다.
    • 범위 - 해당 API로 호출할 수 있는 프로젝트, 워크플로우, 모델을 지정합니다. 다중 선택이 가능합니다.
    • 추가 - 범위 설정을 추가합니다.

4. More Actions 버튼​

  • 편집
    • API 키 관리 화면을 편집모드로 이동합니다.
  • Key 복사
    • Key ID와 Key를 확인하고 복사할 수 있는 팝업이 나타납니다.
  • Token 발급
    • API 호출을 위한 토큰을 발급합니다.
      • 영구 토큰
      • 만료시간 설정(5분, 10분, 30분, 60분, 1일)
    • 발급된 토큰 정보를 표시합니다.
  • API 목록
    • API Key에 지정된 서비스 정보를 표시합니다. 매뉴얼로 링크를 제공합니다.
  • Key 재발급
    • API Key를 재발급합니다.
  • 삭제
    • API Key를 삭제합니다.
