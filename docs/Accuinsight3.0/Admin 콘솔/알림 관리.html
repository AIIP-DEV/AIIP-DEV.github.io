알림 관리

서비스 이용 시 Slack을 통한 알림 관리를 수행합니다.

알림 관리 화면​

1. 알림 수신 목록​

  • 이름
    • 알림 이름
  • URL
    • 알림 수신 URL
  • Message
    • 알림 메시지

2. RUN​

  • 지정된 URL로 메시지가 정상적으로 전달되는지 테스트를 수행합니다.

3. 알림 수신 삭제​

  • 등록된 알림 수신을 삭제합니다.

4. 알림 추가​

  • 알림 수신을 추가합니다.
    • 이름
      • 알림 수신을 구분하는 이름을 입력합니다.
      • 필수 항목
    • URL
      • 알림 메시지를 수신할 URL을 입력합니다.
      • 필수 항목
    • Message
      • 전달할 메시지를 JSON 포맷으로 입력합니다.
      • 필수 항목

5. 수정​

  • 알림 추가 시 필수 항목을 입력한 후 RUN을 수행하여 알림 수신이 성공적으로 수행되어야 수정 버튼이 활성화됩니다.
  • 수정 버튼을 클릭하여 추가한 알림을 저장합니다.
