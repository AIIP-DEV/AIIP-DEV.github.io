자원 승인 내역 관리

프로젝트 생성/수정 및 모델 배포시에 요청한 자원 요청 내역을 확인하고, 관리합니다.

자원요청 내역​

  • 1.필터
    • 검색 가능 필터(날짜 검색 부분은 결재 요청일 기준으로 필터링)
  • 2.승인상태
    • 승인: 초록색 동그라미로 표시
    • 반려: 빨간색 동그라미로 표시 (※승인된 프로젝트의 자원 변경 요청에 경우엔 속이 비어있는 초록색 동그라미로 표시)
    • 대기중: 회색 동그라미로 표시 (※승인된 프로젝트의 자원 변경 요청에 경우엔 속이 비어있는 초록색 동그라미로 표시)
    • 취소: 빨간색 동그라미로 표시(승인 요청한 프로젝트가 삭제된 경우)
  • 3.결재 요청일
    • 결재 요청을 보낸 날짜
  • 4.결재 승인일
    • 승인 날짜

프로젝트 자원승인요청 상세 팝업​

  1. 프로젝트 명 및 자원요청 정보
  2. 자원요청 명
  3. 자원에 사용되는 클러스터 명
  4. 요청한 학습 자원
  5. 요청한 학습 GPU 자원
  6. 요청한 배포 자원
  7. 요청한 배포 GPU 자원
  8. 승인자 Comment

로그인한 사용자에게 온 요청일 경우 결재 Comment 입력 및 반려/승인 가능

모델 배포 자원승인요청 상세 팝업​

  1. 프로젝트 명 및 자원요청 정보
  2. 자원요청 명
  3. 모델이 소속된 프로젝트 / 워크플로우
  4. 모델 버전 설명
  5. 모델 프레임워크 버전
  6. 데이터 타입
  7. 모델 경로
  8. 사용자 코드
  9. 학습에 사용된 데이터
  10. 승인자 코멘트

로그인한 사용자에게 온 요청일 경우 결재 Comment 입력 및 반려/승인 가능
